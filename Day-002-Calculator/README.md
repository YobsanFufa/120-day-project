A robust command-line calculator built with Python.
Part of the **120 Days of Code Challenge** - Day 2.

## Features
- **Basic Arithmetic**: Addition, Subtraction, Multiplication, Division.
- **Advanced Math**: Powers and Square Roots.
- **Error Handling**: gracefully handles division by zero and valid inputs.
- **History**: Automatically logs all calculations to `history.txt`.

## Usage
Run the main script:
```bash
python main.py
```
Type the name of the operation (e.g., `add`, `sqrt`) when prompted.

## Project Structure
- `main.py`: User interface and input handling.
- `operations.py`: Pure math functions.
- `history.txt`: Log file.
