# 120 Days of Code Challenge

This repository logs my journey through the **120 Days of Code** challenge.

## Rules
1.  **Code Daily**: Spend time coding every single day.
2.  **Log Progress**: Update `challenges.md` with daily activities.
3.  **Push to GitHub**: Commit and push changes daily.

## Structure
- `challenges.md`: The main tracker file containing a daily log table.
- Projects will be organized in folders or linked if external.

## Tools
- **Assistant**: Agentic AI (Antigravity)
- **Editor**: VS Code / IDX
- **Version Control**: Git
